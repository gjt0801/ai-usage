# Codex Prompt: Implementation Brief

Use this when asking Codex to turn an idea into code.

```text
Goal:
<what should change?>

Context:
<repo, product, user, or technical background>

Constraints:
<must keep, must avoid, compatibility, deadlines>

Expected output:
- implement the change
- run relevant checks
- summarize changed files
- list any remaining risks

Please inspect the repo first, then implement the smallest useful version.
```
