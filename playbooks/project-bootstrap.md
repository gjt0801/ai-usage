# Project Bootstrap Playbook

Use when starting or importing a project.

## Steps

1. Define the repo purpose in `README.md`.
2. Add a minimal directory structure.
3. Preserve third-party licenses when importing resources.
4. Add `.gitignore` before generated files appear.
5. Make one clear initial commit per logical import.

## AI Cycle Setup

- Put workflow docs under `workflows/`.
- Put reusable prompts under `prompts/`.
- Put external references under `vendor/`.
- Put lessons and experiment notes under `logs/experiments/`.
